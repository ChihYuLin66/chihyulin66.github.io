server {

    listen 80;
    listen [::]:80;

    # (1) 網址
    server_name laravel7-30days.test;

    # (2)專案路徑
    root /var/www/html/laravel7-30days/public;
    index index.php index.html index.htm;

    location / {
         try_files $uri $uri/ /index.php$is_args$args;
    }

    location ~ \.php$ {
        try_files $uri $uri/ =404;

        # (3) php使用
        # local
        # fastcgi_pass unix:/var/run/php/php7.3-fpm.sock;

        # docker
        fastcgi_pass PHP7.3:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;

        # for phpstorm / xdebug
        #fastcgi_param PHP_IDE_CONFIG serverName=localhost;
        fastcgi_buffers 8 128k;
        fastcgi_buffer_size 256k;

        # for long running debug sessions
        fastcgi_read_timeout 600;
        #fastcgi_intercept_errors on;

        include fastcgi_params;
    }

    location ~ /\.ht {
        deny all;
    }
}

<VirtualHost *:80>
    # (1) 專案路徑
    DocumentRoot "/var/www/html/laravel7-30days/public"

    # (2) 網址
    ServerName laravel7-30days.test

    # (3) LOG的地方
    ErrorLog "/var/www/html/laravel7-30days/storage/laravel7-30days_error.log"
    CustomLog "/var/www/html/laravel7-30days/storage/laravel7-30days_access.log" combined
    DirectoryIndex index.php

    # (4) 專案路徑
    <Directory "/var/www/html/laravel7-30days/public">
        AllowOverride All
        Order allow,deny
        Allow from all
    </Directory>
</VirtualHost>

其他

以下設定我都是使用預設值，
但還是簡單跟大家介紹一下

  1. 隊列設定，可以參考 /config/queue.php 設定，預設是 sync，
    如寄信功能就可能需要設定，避免執行過程中畫面卡著。
    1

    QUEUE_CONNECTION=sync
  2. Broadcaster 廣播，可以發通知的功能
    可以參考 /config/broadcasting.php (這個我目前沒用過)
    1

    BROADCAST_DRIVER=log
  3. 快取設定，可以參考 /config/cache.php
    1

    CACHE_DRIVER=file
  4. Seesion 設定，可以參考 /config/session.php
    1
2

    SESSION_DRIVER=file
SESSION_LIFETIME=120

結語

env 的設定，都是跟 /config/ 底下檔案息息相關，
可以理解成 config 是預設值， env是設定值
所以我開發時都會直接用 config() 調用他們。

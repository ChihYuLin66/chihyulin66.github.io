如何在 Linux 上安裝 Homebrew ?

Dec 10, 2024
1 minute read

什麼是 Homebrew？

Homebrew 是一個開源的套件管理工具，可以輕鬆地安裝、更新、刪除和管理軟體或開發工具。
在 Linux 上，Homebrew 也能幫助你處理系統的依賴性，簡化套件的管理流程。

安裝前的準備

在開始之前，請確保你的 Linux 系統具有以下條件：

  • 支援的 Linux 發行版（如 Ubuntu、Debian 等）。
  • 安裝了基本的系統工具（如 bash 和 curl）。

安裝步驟

安裝 Homebrew

執行以下命令來安裝 Homebrew：

1

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

將 Homebrew 加入 PATH

安裝完成後，將 Homebrew 加入系統的 PATH 以便隨時使用。

執行以下命令來修改你的 .bashrc 文件：

1
2
3

echo >> /home/你的使用者名稱/.bashrc
echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> /home/你的使用者名稱/.bashrc
eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"

這些命令會將 Homebrew 的路徑加到你的環境變數中，讓系統能正確找到 brew 命令。

安裝 Homebrew 的依賴

這個指令會安裝一組編譯工具，確保 Homebrew 可以正常運行。

1

sudo apt-get install build-essential

安裝 GCC 編譯器

Homebrew 依賴 GCC（GNU 編譯器集合）來構建一些套件。

你可以透過以下指令安裝：

1

brew install gcc

測試

確認 brew 是否正常運行

1

brew help

常見指令

搜索套件：

1

brew search [套件名稱]

安裝套件：

1

brew install [套件名稱]

更新 Homebrew：

1

brew update

移除套件：

1

brew uninstall [套件名稱]

檢查系統狀態：

1

brew doctor
